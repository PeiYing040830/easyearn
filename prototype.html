Prototype Objective Coverage

Objective 1

Multi-user portal based on the real EasyEarn roles: Job Seeker, Employer, and Admin.

CRUD: Register user, edit profile, create/update/delete job post

Objective 3

Verifiable worker identity using application status, completed work history, and generated resume identity.

Flow: Apply -> Accepted -> Completed -> Work History

Objective 4

Multilingual accessibility mock using a language selector saved in localStorage.

Language demo: English interface selected.

Based on Existing EasyEarn System

This prototype uses localStorage, but the screens follow the same modules and data flow as the Supabase system.

usersregistration, login, role, seeker profile, employer verification
job_listingsemployer job CRUD, admin approval, public job browsing
applicationsjob seeker apply, employer review, accept, complete
saved_jobssave and remove bookmarked job
work_historycompleted work record for digital identity
prototype scopereports, ratings, payments, analytics, and chatbot are excluded from this basic version
