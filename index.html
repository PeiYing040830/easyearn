Project Coverage

EasyEarn Coverage & User Types

Multi-user web portal with role-based access and tailored functionality.

Job Seekers

  • Register and build profiles with skill tags
  • Browse and filter jobs by category and location
  • Apply to jobs and track application status
  • Save jobs to wishlist and view work history
  • Auto-generate PDF resume from work history
  • Use chatbot for 24/7 support

Employers

  • Register and obtain verification badge
  • Create and manage job listings with expiry dates
  • Review applications and approve or reject candidates
  • Rate job seekers after completion

Administrators

  • User account control and role management
  • Job listing moderation and enforcement
  • Report and flag resolution
  • Platform analytics and monitoring
System Modules

Core Modules of EasyEarn

Each module supports a critical step in the job matching lifecycle.

1. User Management

Registration, login, and role-based access control for Job Seekers, Employers, and Administrators.

2. Job Posting & Application

Employers post jobs with category, location, salary, and expiry; seekers apply and track status.

3. Safety & Trust

Employer verification badges, report and flag system, and bidirectional ratings.

4. Work History & Resume

Track completed jobs, earnings, job type distribution, and auto-generate PDF resumes.

5. Chatbot & Support

Rule-based 24/7 assistance with FAQ guidance and application help.

6. Admin & Analytics

Admin dashboard for platform control and analytics powered by Chart.js.

7. Google Translate Integration

Language support through Google Translate redirection for Bahasa Malaysia, Mandarin, and Tamil.

Security

Information Security & Policies

Aligned with Malaysian regulations and technical safeguards.

Regulatory Framework

  • Personal Data Protection Act (PDPA) 2010
  • Computer Crimes Act 1997
  • Consumer Protection Act 1999
  • Employment Act 1955 (Reference Only)

Technical Security Measures

  • Supabase authentication with secure session handling
  • Role-Based Access Control (RBAC)
  • HTTPS encryption for all data transmission
  • Row Level Security (RLS) policies to enforce access
